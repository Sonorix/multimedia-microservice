Multimedia Microservice

Manage musician profiles, ratings, and multimedia content

Service Status

Active and running

API Base Path: /Multimedia-microservice/resources

API Overview

Musician Profiles

GET /profiles

Get all musician profiles

GET /profiles/{id}

Get musician profile by ID

GET /profiles/user/{userId}

Get musician profile by user ID

POST /profiles

Create a new musician profile

PUT /profiles/{id}

Update a musician profile

DELETE /profiles/{id}

Delete a musician profile

Ratings

GET /ratings/musician/{musicianId}

Get all ratings for a musician

GET /ratings/{id}

Get a specific rating by ID

GET /ratings/musician/{musicianId}/user/{userId}

Get rating from a specific user for a musician

POST /ratings

Add a new rating for a musician

DELETE /ratings/{id}

Delete a rating

Multimedia Files

GET /multimedia/musician/{musicianId}?publicOnly=true|false

Get all multimedia files for a musician

GET /multimedia/{id}

Get metadata for a specific file

GET /multimedia/{id}/download

Download a file

POST /multimedia/upload

Upload a new file (multipart/form-data)

PUT /multimedia/{id}

Update file metadata

DELETE /multimedia/{id}

Delete a file
